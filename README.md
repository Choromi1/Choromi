# Valheim 서버 모드 가이드

업로드된 `config.zip` 기준으로 실제 설정값 설명을 포함한 GitHub Pages용 정적 웹페이지입니다.

## 배포
저장소 루트에 아래 파일을 업로드하세요.

```text
index.html
styles.css
script.js
config_summary.json
README.md
```

## 프로필 코드 수정
`index.html`에서 아래 문구를 실제 r2modman 프로필 코드로 교체하세요.

```text
여기에 r2modman 프로필 코드를 입력하세요
```

## 비주얼 모드 수동 설치 기준
r2modman에서 프로필 선택 후 `Settings → Browse profile folder`로 프로필 폴더를 열고 안내 경로에 복사하세요.
